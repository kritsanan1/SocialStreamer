## [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#using-the-dashboard)  Using the Dashboard

If your Ayrshare account is entitled to manage multiple profiles, then you will see the option for **User Profiles** in the left-hand navigation.

If your account does not have the option to manage profiles, then contact your Ayrshare
representative or [contact us](mailto:contact@ayrshare.com) to upgrade.

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#create-a-new-profile)  Create a New Profile

Click the **Add Profile** button and enter your User Profile Title.![Create a User Profile](https://mintlify.s3.us-west-1.amazonaws.com/ayrshare-docs/images/multiple-users/create-profile.webp)

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#switch-to-the-new-profile)  Switch to the New Profile

You will see the new Profile show up in the Active Profiles section.
You can now click on the **switch to this profile** icon to switch to that User Profile.![Switch to a User Profile](https://mintlify.s3.us-west-1.amazonaws.com/ayrshare-docs/images/multiple-users/switch-profile.webp)When you switch to a different User Profile, you will see that profile name shown in the top left of the screen under the profile icon.You can always switch back to your primary account profile by clicking the **switch profile** link next to the Primary Profile or by clicking the **Switch to Primary** button at the top of the User Profiles page.From here you are acting as your client/user and can post on their behalf by going the the _Post_ page.

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#get-the-profile-key)  Get the Profile Key

The Profile Key is used to post on behalf of the profile with the API. Switch to a [User Profile](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles) and go to the _Profile Key page_ in the left-hand navigation of the Dashboard to get that profile’s Key.
When making an API call, be sure to include your [API\_KEY](https://www.ayrshare.com/docs/apis/overview#authorization) and [PROFILE\_KEY](https://www.ayrshare.com/docs/apis/overview#profile-key-format) in the header.You also receive the Profile Key when you create a new User Profile using the [/profiles/profile endpoint](https://www.ayrshare.com/docs/apis/profiles/create-profile). This is the preferred method to get the Profile Key.

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#invite-a-team-member)  Invite a Team Member

You can create Users Profiles for members of your team. By default, Team Members have an Ayrshare account with full access/admin rights to the dashboard and count as a new User Profile. Please note, you may restrict access by [locking the Primary Profile](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles#primary-profile-lock).

You can only invite team members who have not yet been registered with Ayrshare. If you want to
use an already registered email, please have the user login to the dashboard, go to the Account
page, and click the **Delete** button.

In the User Profiles page, switch to your Primary Profile and click the _\+ Add Profile_ button at the top of the page.In the pop-up modal, click the checkbox “Create a team member profiles with admin rights” box and enter your team member’s email address.![Invite team member](https://mintlify.s3.us-west-1.amazonaws.com/ayrshare-docs/images/multiple-users/team-member.webp)Click _Add Team Profile_ and an email is sent to the team member’s email address with a link to activate their account.

#### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#team-member-status)  Team Member Status

Once the team member has accepted the invite you will see a green check mark next to the Team Member tag.![team member accepted](https://mintlify.s3.us-west-1.amazonaws.com/ayrshare-docs/images/multiple-users/team1.webp)If the team member has not yet accepted the invite you may resend it by clicking the mail icon.![team member resend](https://mintlify.s3.us-west-1.amazonaws.com/ayrshare-docs/images/multiple-users/team2.webp)

#### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#team-members-contacting-support)  Team Members Contacting Support

For security, only registered team members may contact support on behalf of a business. We will
not be able to assist if they are not registered and verified as a team member of your business.

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#reactivate-a-suspended-user-profile)  Reactivate a Suspended User Profile

#### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#suspension-reason)  Suspension Reason

When a User Profile gets suspended due to a social network violation (such as a [Facebook community standards violation](https://www.ayrshare.com/docs/help-center/technical-support/facebook_or_instagram_account_restricted)), you will receive a detailed notification email.
This email contains important information about the suspension, including the specific reason and the affected User Profile.To locate this notification email:

1. Search your inbox using the RefId of the suspended User Profile
2. The email subject will clearly indicate a suspension has occurred

Alternatively, you can view suspension details directly in the dashboard by clicking on the “Suspended” tag that appears next to the affected User Profile.All API calls for that User Profile will return a [403 HTTP status code](https://www.ayrshare.com/docs/errors/errors-http#403-access-denied).

#### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#reactivate-a-suspended-user-profile-2)  Reactivate a Suspended User Profile

You may reactivate, i.e. unsuspend, the User Profile by:

1. Logging into the Dashboard and going to the User Profiles page.
2. Find the suspended User Profiles by clicking the “Suspended” tab or searching by the RefId of the User Profile, which is included in the email.
3. Unsuspend the User Profile by clicking the “Unsuspended” button.
4. Review and accept the compliance certification.

![Unsuspend a User Profile](https://mintlify.s3.us-west-1.amazonaws.com/ayrshare-docs/images/multiple-users/suspended.webp)For a User Profile’s first suspension, the User Profile can be immediately unsuspended once you accept the compliance certification.
However, for subsequent suspensions the User Profile will be reinstated _48 hours_ after accepting the compliance certification.

We recommend you _do not_ use your Primary Profile as a regular User Profile. If the Primary
Profile is suspended, all user profiles will also be suspended.

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#managing-active-user-profiles)  Managing Active User Profiles

Effective management of User Profiles is essential for maintaining an efficient system in Ayrshare.
We recommend implementing a policy to remove inactive User Profiles based on your business requirements.
For example, if a user hasn’t engaged with your platform by posting content, requesting analytics for some time period, or has not connected any social networks, consider deleting their Ayrshare User Profile, thus keeping your user base current and reducing future costs.You can check if a User Profile has been active in the current month by reviewing their [posting history](https://www.ayrshare.com/docs/apis/history/overview) and [API usage and connections](https://www.ayrshare.com/docs/apis/user/overview).
If you determined it to be inactive, [delete the User Profile](https://www.ayrshare.com/docs/apis/profiles/delete-profile).

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#instagram-login)  Instagram Login

Enable or disable Instagram login on the social linking page.![Instagram Login](https://mintlify.s3.us-west-1.amazonaws.com/ayrshare-docs/images/multiple-users/instagram-login.webp)Choose whether to enable direct Instagram Login linking that does not require a connected Facebook Page for authentication. If disabled, Instagram linking will require a [connected Facebook Page](https://www.ayrshare.com/docs/dashboard/connect-social-accounts/facebook) for authentication.

Instagram Login is **enabled by default**. If you require the advanced features listed below
(hashtag search, collaborations, location tagging, or brand data), you must manually disable this
option to use Facebook Page authentication instead.

#### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#direct-instagram-login-vs-facebook-page-authentication)  Direct Instagram Login vs Facebook Page Authentication

**Direct Instagram Login** Instagram API with Instagram Login allows Instagram professionals (businesses and creators) to link their accounts directly without needing an associated Facebook Page. This provides a simplified authentication flow.**Facebook Page Authentication** (traditional method) requires users to first connect a Facebook Page that is linked to their Instagram professional account.

#### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#feature-limitations-with-direct-instagram-login)  Feature Limitations with Direct Instagram Login

While direct Instagram Login offers convenience, certain features are **not available** when using this method:

- **[Brand/User Data](https://www.ayrshare.com/docs/apis/brand/brand-user#param-instagram-user)** \- Cannot retrieve Instagram account information through the brand endpoints.
- **[Hashtag Search](https://www.ayrshare.com/docs/apis/hashtags/search-hashtags)** \- Cannot search for hashtags on Instagram.
- **[Collaborations](https://www.ayrshare.com/docs/apis/post/social-networks/instagram#collaboration)** \- Cannot invite collaborators to Instagram posts.
- **[Location Tagging](https://www.ayrshare.com/docs/apis/post/social-networks/instagram#location)** \- Cannot add location tags to Instagram posts.
- **[`idId`\- Legacy Instagram User ID](https://www.ayrshare.com/docs/apis/analytics/social)** \- field not returned for Instagram social analytics.

#### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#which-method-should-you-choose%3F)  Which Method Should You Choose?

- **Enable Direct Instagram Login** if you only need basic posting functionality and don’t require the advanced features listed above.
- **Disable Direct Instagram Login** (use Facebook Page authentication) if you need access to all Instagram features, including hashtag search, collaborations, location tagging, or brand data retrieval.

For most clients, the preferred method is to use the direct Instagram Login authentication. If
you’re unsure which method to use, we recommend disabling Direct Instagram Login to ensure access
to all features. You can always enable it later if you determine the advanced features are not
needed.

### [​](https://www.ayrshare.com/docs/multiple-users/manage-user-profiles\#automatically-resync-facebook-and-instagram-pages)  Automatically Resync Facebook and Instagram Pages

When security events occur on Facebook, such as password changes, all Facebook pages associated with that user account (including linked Instagram accounts) need to be reconnected to Ayrshare. For accounts managing numerous pages, this process can be time-consuming.

- The “Resync” option is available when connecting Facebook or Instagram pages.
- By enabling this feature, all previously connected pages will be automatically relinked to the
user profile.
- Note: Only previously linked pages will be reconnected; new pages will not be added
automatically.
- This feature supports Facebook accounts with up to 400 pages.
- For accounts exceeding 400 pages, it’s recommended to distribute pages across multiple Facebook
accounts.
- **Best Practice:** Only use the “Resync” option when necessary to reconnect all pages. For
routine operations, it’s generally not required.
